A simple plugin to add a subtitle (for a post or a page). You may want to use custom CSS for a better rendering.

== Description ==

This plugin just adds a subtitle to posts and pages. You need to fill the subtitle in a custom field. The subtitle will be displayed just under the title. No additional CSS for now, you can add it as additional CSS.
